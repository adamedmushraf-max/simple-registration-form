# Simple Registration Form

A simple and responsive Registration Form built using **HTML** and **JavaScript**. This project demonstrates the fundamentals of form creation, user input handling, and basic client-side validation.

## Features
- User-friendly registration form
- Input fields for user details
- Basic form validation using JavaScript
- Clean and responsive design
- Beginner-friendly project

## Technologies Used
- HTML5
- JavaScript (ES6)

## Purpose
This project was created to practice HTML form elements, DOM manipulation, event handling, and JavaScript validation techniques.

## Future Improvements
- Add CSS styling for a modern UI
- Store user data using Local Storage
- Improve validation with regular expressions
- Connect the form to a backend database
